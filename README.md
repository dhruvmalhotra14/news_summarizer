# 📑 Professional News Summarizer (Stealth AI)

An AI-powered web application that extracts and summarizes news articles into high-detail reports. 

## 🚀 Key Features
* **Stealth Scraping:** Bypasses security using custom Selenium headers.
* **Deep Analysis:** Forces 15-line professional summaries using FLAN-T5.
* **Smart Categorization:** Automatically labels news as Defense, Economy, or Politics.
